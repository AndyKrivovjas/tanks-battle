# Tanks Battle

Hi there! I am creating a simple tank game for me and my friends.

## How to play
1. Movement
  * **Forward** - arrow up
  * **Backward** - arrow down
  * **Turn Left** - arrow left
  * **Turn Right** - arrow right
2. Shooting 
  * **Shoot** - SPACEBAR
